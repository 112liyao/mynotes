##数据库
* 增
```javascript
    insert  into  <表名>  <字段>  values  <字段值>
```
* 删
```javascript
    delete  from  <表名>  where  <删除的条件>
```
* 改
```javascript
    update   <表名>   set  <字段=字段值>  where   <更新的条件>
```
* 查
```javascript
    select  <字段>  from  <表名>  where  <查询条件>
```
